# Security Policy

## Supported Versions

Only the latest version of **searXena** is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability within this project, please send an e-mail to **neuralarchlabs@gmail.com**. All security vulnerabilities will be promptly addressed.

Please include the following in your report:
- Type of issue (e.g., unauthorized access, data leak, etc.)
- Step-by-step instructions to reproduce the issue.
- Potential impact.

**Do not open a GitHub Issue for security vulnerabilities**, as they are public and can put other users at risk.

*Thank you for helping keep searXena safe.*
